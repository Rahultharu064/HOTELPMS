# Hotel PMS Project Outline

## 1. Core Modules
- **Front Office**: Dashboard, Bookings, Check-in/out, Payments, Reports.
- **Admin**: System configuration, Room Type management, User management, Inventory.
- **Housekeeping**: Room status monitoring, Staff assignment, Cleaning logs.
- **Service POS**: Order management for extra services (linked to bookings or walk-ins).
- **Public Website**: Guest-facing booking engine, Facilities showcase, Reviews.

## 2. Implemented Features
- [x] **Multimedia Room Inventory**: Support for room images and videos.
- [x] **Multichannel Booking**: Online and Offline/Walk-in reservations.
- [x] **Guest ID Verification**: Secure storage of ID documents.
- [x] **Payment Integration**: Support for eSewa, Khalti, and Cash.
- [x] **Service Billing**: POS system for tracking extra services.
- [x] **Housekeeping Workflow**: Cleaning states and staff tracking.
- [x] **Transactional Integrity**: Robust backend logic for room swaps and check-ins.

## 3. Latest Updates
- **Professional Payment Segregation**: Distinct flows for room payments vs service payments.
- **Service POS Page**: New interface for rapid service order processing.
- **Enhanced Verification**: Added identification document validation in the check-in flow.

## 4. Tech Stack
- **Frontend**: React + TS + Vite (Modern Aesthetics).
- **Backend**: Node.js + Express + Prisma.
- **Database**: MySQL.
